python
debugger模式下的命令
  • c: 继续执⾏
  • w: 显⽰当前正在执⾏的代码⾏的上下⽂信息
  • a: 打印当前函数的参数列表
  • s: 执⾏当前代码⾏，并停在第⼀个能停的地⽅（相当于单步进⼊）
  • n: 继续执⾏到当前函数的下⼀⾏，或者当前⾏直接返回（单步跳过）
